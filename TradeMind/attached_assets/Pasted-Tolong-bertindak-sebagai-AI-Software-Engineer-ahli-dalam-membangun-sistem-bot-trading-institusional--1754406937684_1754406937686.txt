### ✅ 3. FUNCTIONAL REQUIREMENTS (WAJIB JALAN)

#### ✅ Platform:
- Windows 10/11 + Python 3.11+
- MetaTrader 5 (MT5) integration
- 1-click python bot.py auto-run

#### ✅ Risk Management:
- TP/SL dihitung dari persentase equity (misal: TP 2%, SL 1% dari modal)
- Hitung lot size otomatis berdasarkan risk %
- Support floating drawdown control & max exposure

#### ✅ Indicators & Technical Analysis:
- *Tanpa TA-Lib!*
- Hitung semua indikator manual: SMA, EMA, WMA, RSI, MACD, ATR, Bollinger Bands, Stochastic, dll.
- Multi-timeframe support (misal Entry di M15, Confirmasi di H1)

#### ✅ Strategy:
- HFT logic: tick-based, sub-millisecond latency, queue jump
- Scalping, swing, pattern
- Multi-symbol (GOLD, BTCUSD, EURUSD, dll)

#### ✅ AI/ML Module:
- Support untuk optional AI-based decision module
- Bisa menggunakan sklearn atau XGBoost untuk prediksi arah pasar

#### ✅ GUI:
- PyQt5 based GUI
- Menampilkan: chart, running balance, equity, open trades, drawdown
- Tombol emergency stop

#### ✅ Sentiment Analysis:
- Analisis berita dan sosial media sederhana (gunakan transformers atau vaderSentiment)

#### ✅ Alert System:
- Telegram API: kirim pesan saat posisi open/close, SL hit, TP hit, DD besar

#### ✅ Auto Connection:
- Tidak perlu login manual
- Saat bot dijalankan, langsung connect MT5 dan jalankan semua modul otomatis
